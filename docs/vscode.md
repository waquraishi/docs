# VSCode Keyboard Commands

⌘T: Show all Symbols
⌘ Shift P: Quick Open Dialog

⌘ P:  Search Files By Name 
⌘ Shift T: Re-open closed terminal
⌘ X: Delete A line 

⌘K: Enter Keep preview mode editor open
